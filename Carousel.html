Instructions d'utilisation

Pour le menu :

Veuillez remplir le formulaire semaine par semaine.

Un indice en bas de la page s'incrémentera à chaque envoi du formulaire afin que vous puissiez savoir à quel jour vous en êtes.

Veillez aussi à supprimer le fichier Menus.JSON avant de remplir le formulaire pour la semaine.

Pour ce faire rendez-vous dans l'onglet Supprimer fichiers


Pour les activités :

Veillez à supprimer le fichier Activite.JSON avant de remplir le formulaire pour les activités.

Pour ce faire rendez-vous dans l'onglet Supprimer fichiers


Pour les rendez-vous :

Veillez à supprimer le fichier RDV.JSON avant de remplir le formulaire pour les rendez-vous.

Pour ce faire rendez-vous dans l'onglet Supprimer fichiers

Supprimer les fichiers .JSON



Pour supprimer un ou plusieurs fichiers, veuillez cocher la ou les cases correspondant aux fichiers à supprimer.
